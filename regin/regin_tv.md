# 获取分区列表

> https://api.snm0516.aisee.tv/x/tv/region/list

请求方式：`GET`

是否需要登录：`否`

## URL参数

| 参数名        | 类型  | 必填  | 内容  | 备注  |
|------------|-----|-----|-----|-----|
| access_key | str |     |     |     |
| build      | num |     |     |     |

## Json回复

### 根对象

| 字段名                   | 类型    | 内容       | 备注   |
|-----------------------|-------|----------|------|
| code                  | num   | 响应码      | 0：成功 |
| data                  | array | 数据本体     |      |
| default_regions       | array | 默认分区     |      |
| default_regions_title | str   | `默认首页分区` |      |
| is_smart              | num   | `0`      |      |
| message               | str   | 0        |      |
| mtime                 | num   |          |      |

### `data`数组中的对象

| 字段名               | 类型   | 内容          | 备注          |
|-------------------|------|-------------|-------------|
| style_id          | num  |             |             |
| type_id           | num  |             |             |
| id                | num  |             |             |
| real_id           | num  |             |             |
| name              | str  |             |             |
| index_tid         | num  |             |             |
| index_type        | num  |             |             |
| type              | num  |             |             |
| image             | str  |             |             |
| image_selected    | str  |             |             |
| image_selected_v2 | str  |             |             |
| image_focus       | str  |             |             |
| image_focus_v2    | str  |             |             |
| image_background  | str  | 分区背景图       |             |
| image_size        | num  | `1`,`2`,`3` |             |
| locked            | num  | 是否锁定不允许移动   | 0：否<br/>1：是 |
| style_type        | num  |             |             |
| is_default        | num  | 是否默认显示      | 0：否<br/>1：是 |
| feed_id           | num  |             |             |
| spmid             | str  |             |             |
| tag_id            | num  |             |             |
| server_info       | str  | `空串`        |             |
| is_fourk          | num  | `0`         |             |
| region_type       | num  | `0`,`1`,`2` |             |
| region_tid        | num  |             |             |
| manual_idx        | num  |             |             |
| rank              | num  |             |             |
| ug_id             | num  |             |             |
| build_limit       | str  |             |             |
| display_url       | str  | `空串`        |             |
| keep_cache        | bool |             |             |
| hot_update        | bool | `false`     |             |
| brand_white_list  | str  |             |             |

### `default_regions`数组中的对象

| 字段名     | 类型   | 内容   | 备注  |
|---------|------|------|-----|
| name    | str  | 分区名称 |     |
| real_id | num  | id   |     |
| default | bool | 是否默认 |     |

## 请求示例

```shell
curl -L -X GET 'https://api.snm0516.aisee.tv/x/tv/region/list?access_key=********************************&build=105400'
```
